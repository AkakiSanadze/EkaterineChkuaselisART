# Ekaterine Chkuaseli's Art - рЃАрЃљрЃўрЃбрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЊрЃљ рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃўрЃА рЃњрЃћрЃњрЃЏрЃљ

## ­ЪЊі рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў

### рЃАрЃљрЃўрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

### РюЁ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃўрЃбрЃўрЃА рЃФрЃџрЃўрЃћрЃарЃў рЃЏрЃ«рЃљрЃарЃћрЃћрЃЉрЃў

1. **рЃћрЃАрЃЌрЃћрЃбрЃўрЃЎрЃБрЃарЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў**
   - рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ, рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃўрЃАрЃбрЃБрЃарЃў рЃАрЃбрЃўрЃџрЃў
   - рЃЌрЃЉрЃўрЃџрЃў рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃърЃљрЃџрЃўрЃбрЃарЃљ (#a67c52 - рЃЏрЃЮрЃюрЃљрЃфрЃарЃўрЃАрЃцрЃћрЃарЃў/рЃЮрЃЦрЃарЃЮрЃАрЃцрЃћрЃарЃў)
   - рЃЎрЃљрЃарЃњрЃў рЃбрЃўрЃърЃЮрЃњрЃарЃљрЃцрЃўрЃљ (Raleway рЃцрЃЮрЃюрЃбрЃў)

2. **рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃЮрЃЉрЃљ**
   - рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃарЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў (рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ)
   - Hero slider рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
   - Lightbox рЃњрЃљрЃџрЃћрЃарЃћрЃљрЃерЃў
   - рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ
   - Sticky navigation

3. **рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў**
   - 70+ рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃАрЃБрЃарЃљрЃЌрЃў
   - рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃљрЃарЃќрЃћ (рЃАрЃљрЃ«рЃћрЃџрЃў, рЃбрЃћрЃЦрЃюрЃўрЃЎрЃљ, рЃќрЃЮрЃЏрЃљ, рЃгрЃћрЃџрЃў)
   - рЃЏрЃ«рЃљрЃбрЃЋрЃарЃўрЃА рЃЉрЃўрЃЮрЃњрЃарЃљрЃцрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ

### РЮї рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃБрЃАрЃбрЃў рЃЏрЃ«рЃљрЃарЃћрЃћрЃЉрЃў

#### 1. рЃЎрЃЮрЃЊрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃљрЃфрЃўрЃљ
- **CSS рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃљ**: рЃДрЃЋрЃћрЃџрЃљ HTML рЃцрЃљрЃўрЃџрЃерЃў inline CSS-рЃћрЃЉрЃўрЃљ рЃњрЃљрЃгрЃћрЃарЃўрЃџрЃў, рЃарЃљрЃф рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃА
- **рЃцрЃБрЃбрЃћрЃарЃўрЃА рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃљ**: рЃўрЃЊрЃћрЃюрЃбрЃБрЃарЃў рЃцрЃБрЃбрЃћрЃарЃўрЃА рЃЎрЃЮрЃЊрЃў 4-рЃЋрЃћ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћрЃљ рЃЊрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃБрЃџрЃў
- **JavaScript рЃњрЃљрЃцрЃљрЃюрЃбрЃБрЃџрЃўрЃљ**: рЃДрЃЋрЃћрЃџрЃљ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ inline JS-рЃћрЃЉрЃўрЃљ

#### 2. SEO рЃЊрЃљ Meta рЃбрЃћрЃњрЃћрЃЉрЃў
- **lang="en"** - рЃАрЃљрЃўрЃбрЃў рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ/рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃљрЃЊрЃљрЃљ
- **рЃљрЃа рЃљрЃарЃўрЃА meta description** - SEO-рЃАрЃЌрЃЋрЃўрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў
- **рЃљрЃа рЃљрЃарЃўрЃА Open Graph рЃбрЃћрЃњрЃћрЃЉрЃў** - рЃАрЃЮрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃЏрЃћрЃЊрЃўрЃўрЃА рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- **рЃљрЃа рЃљрЃарЃўрЃА structured data (Schema.org)**

#### 3. рЃърЃћрЃарЃцрЃЮрЃарЃЏрЃљрЃюрЃАрЃў
- **рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃў** - рЃЊрЃўрЃЊрЃў рЃќрЃЮрЃЏрЃўрЃА JPG рЃцрЃљрЃўрЃџрЃћрЃЉрЃў
- **рЃљрЃа рЃљрЃарЃўрЃА lazy loading** - рЃДрЃЋрЃћрЃџрЃљ рЃАрЃБрЃарЃљрЃЌрЃў рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ
- **рЃљрЃа рЃљрЃарЃўрЃА CSS/JS minification**

#### 4. рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃљ (Accessibility)
- **рЃљрЃа рЃљрЃарЃўрЃА skip-to-content рЃЉрЃЏрЃБрЃџрЃў**
- **рЃќрЃЮрЃњрЃўрЃћрЃарЃЌ рЃАрЃБрЃарЃљрЃЌрЃА рЃљрЃЎрЃџрЃўрЃљ alt рЃбрЃћрЃЦрЃАрЃбрЃў**
- **рЃЎрЃЮрЃюрЃбрЃарЃљрЃАрЃбрЃўрЃА рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў** рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА
- **рЃљрЃа рЃљрЃарЃўрЃА ARIA labels** рЃДрЃЋрЃћрЃџрЃљ рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃа рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ

#### 5. UX/UI рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃћрЃЉрЃў
- **рЃљрЃа рЃљрЃарЃўрЃА breadcrumbs**
- **рЃљрЃа рЃљрЃарЃўрЃА рЃФрЃћрЃЉрЃюрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ** рЃњрЃљрЃџрЃћрЃарЃћрЃљрЃерЃў
- **рЃљрЃа рЃљрЃарЃўрЃА "рЃќрЃБрЃарЃњрЃўрЃЌ рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃў** рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃа рЃ«рЃћрЃЊрЃерЃў
- **рЃљрЃа рЃљрЃарЃўрЃА pagination** рЃњрЃљрЃџрЃћрЃарЃћрЃљрЃерЃў (рЃДрЃЋрЃћрЃџрЃљ рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃљрЃарЃў рЃћрЃарЃЌ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћрЃљ)
- **рЃАрЃЮрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃў placeholder-рЃћрЃЉрЃўрЃљ** (#)

#### 6. рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў
- **рЃљрЃа рЃљрЃарЃўрЃА рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў** - lightbox-рЃерЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃАрЃБрЃарЃљрЃЌрЃўрЃљ
- **рЃљрЃа рЃљрЃарЃўрЃА "рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃћрЃЉрЃў"** рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ
- **рЃљрЃа рЃљрЃарЃўрЃА рЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ** рЃљрЃю "рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА" рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ
- **рЃљрЃа рЃљрЃарЃўрЃА newsletter signup**

## ­Ъј» рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃўрЃА рЃњрЃћрЃњрЃЏрЃљ

### рЃцрЃљрЃќрЃљ 1: рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃќрЃљрЃфрЃўрЃљ (рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃцрЃБрЃФрЃћ)

#### 1.1 CSS рЃЏрЃЮрЃЊрЃБрЃџрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- рЃерЃћрЃЦрЃЏрЃћрЃюрЃў `css/` рЃцрЃЮрЃџрЃЊрЃћрЃарЃў
- рЃњрЃљрЃўрЃбрЃљрЃюрЃћ рЃАрЃбрЃўрЃџрЃћрЃЉрЃў рЃфрЃљрЃџрЃЎрЃћ рЃцрЃљрЃўрЃџрЃћрЃЉрЃерЃў:
  - `main.css` - рЃАрЃљрЃћрЃарЃЌрЃЮ рЃАрЃбрЃўрЃџрЃћрЃЉрЃў
  - `header.css` - рЃ░рЃћрЃЊрЃћрЃарЃў
  - `footer.css` - рЃцрЃБрЃбрЃћрЃарЃў
  - `gallery.css` - рЃњрЃљрЃџрЃћрЃарЃћрЃљ
  - `responsive.css` - рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў

#### 1.2 JavaScript рЃЏрЃЮрЃЊрЃБрЃџрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- рЃерЃћрЃЦрЃЏрЃћрЃюрЃў `js/` рЃцрЃЮрЃџрЃЊрЃћрЃарЃў
- рЃњрЃљрЃўрЃбрЃљрЃюрЃћ JS рЃфрЃљрЃџрЃЎрЃћ рЃцрЃљрЃўрЃџрЃћрЃЉрЃерЃў:
  - `main.js` - рЃАрЃљрЃћрЃарЃЌрЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў
  - `gallery.js` - рЃњрЃљрЃџрЃћрЃарЃћрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў
  - `slider.js` - рЃАрЃџрЃљрЃўрЃЊрЃћрЃарЃў
